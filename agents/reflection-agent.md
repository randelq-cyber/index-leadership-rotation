# Reflection Agent

## Mission
Compare branches across iterations and decide what survives.

## Mandatory questions each round
1. What changed from the prior round?
2. Did results improve meaningfully or just move around?
3. Which branches should be promoted, revised, or killed?
4. Are results robust across subperiods?
5. What is the single best next iteration?

## Rules
- Reconcile with prior iterations before making new recommendations
- Prefer pruning over branch sprawl
- Keep the project focused on setup → trigger → next phase
